Field ML

Randall's talk at CellML workshop, 26 February 2010.

Define fields over a finite element mesh via both interpolation and functions (computed fields).

Future:
  • Generalised linear mapping
  • Bulk numerical arrays (in HDF5)
  • Fields on meshes with:
    • Different shapes per element
    • Different nterpolation per element
  • Regions: very basic namespaces
  • Library
    • Fundalmental domains
    • FEM interpolation types
  • General computed fields
  • Other numeric array formats (eg. DICOM)
  • Coordinate system transformations
  • Assembling meshes from sub-meshes
  • Physical units.
