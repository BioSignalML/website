Point

A single datum of a signal. Can have multiple components.

Type of a component:
  • Boolean
  • Integer
  • Rational
  • Real
  • Complex
Also Natural Number, Positive Integer, etc as subsets of Integer. Ditto for Non-Negative Real etc.

The format used for computer representation of the value of a particular type is a separate issue -- eg. A Real value could be stored as a 32-bit IEEE float, a 64-bit IEEE double, etc, etc.
