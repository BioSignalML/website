Partial BioSignalML Implementations

This is when we have a signal recording file without a metadata repository.

The intent is to use the same set of API calls to access signals and metadata contained in the file.
val file = new File(filename)
val rec = new Recording(file)   // overloaded constructor --> Recording(file.toURI, None)
val format = rec.getFormat()
val sig1 = rec.getSignal("1")
versus:
val repo = new jenaRepository(location)
val rec = new Recording(recuri, repo)
// Do we find file as { recuri  DC:source  ?file } and open it?? NO
// or does recuri.toURL() locate the file?                            YES
// and what if the file doesn't exist? Or is in the wrong format??
